Sistema completo para controle de estoque e associação de produtos e matérias-primas.  
Projeto desenvolvido como teste prático, com **backend em FastAPI** e **frontend em React + TailwindCSS**.

---

## 📂 Estrutura do Projeto

- `backend/` – API REST para CRUD de produtos e matérias-primas, associação entre eles, validação e documentação via Swagger  
- `frontend/` – Interface web para gerenciamento de produtos e matérias-primas, integração com a API backend
